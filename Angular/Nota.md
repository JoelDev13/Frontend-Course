## Que es Angular
Es un framework de desarrollo web desarrollado por Google que se utiliza para crear aplicaciones web modernas, en particular las de una sola página (SPA). Sirve para crear interfaces web dinámicas, rápidas y escalables

## Componente
Son los bloques principales de construcción de las aplicaciones Angular. Cada componente representa una parte de una página web como pequeño bloques

## Señales (Signals)
Es una forma nueva y más simple de manejar el estado y la reactividad dentro de una app

### señales computadas
son signals de solo lectura que derivan su valor de otras signal

## Pipe
Es un Objeto que permite transformar de manera visual la información que mostraremos en pantalla sin mutar la data real
